# Personal Dashboard

A personal dashboard app built with React during a 5-day spring break sprint.

**Live demo:** https://react-dashboard-nu-ebon.vercel.app/

## Features

- **Live clock** — updates every second using `useEffect` + `setInterval`
- **Weather widget** — fetches current conditions for Davis, CA from OpenWeatherMap API
- **Quote widget** — loads a random inspirational quote on page load with a refresh button
- **Todo list** — add, check, and delete tasks, persisted in `localStorage`

## Tech Stack

- React 18 (Vite)
- Plain CSS
- OpenWeatherMap API
- DummyJSON Quotes API
- Deployed on Vercel

## Run locally
```
npm install
npm run dev
```

## What I learned

Built this as my first React project. Practiced components, props, useState, useEffect,
API fetching with loading/error states, and localStorage persistence.
